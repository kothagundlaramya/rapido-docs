# Common Issues While Booking Rapido Ride

This page lists common problems faced by users and how to fix them.

## Issue 1: App not opening
**Possible reason:**
- Poor internet connection

**Solution:**
- Turn ON mobile data or Wi-Fi
- Restart the app

## Issue 2: Location not detected
**Possible reason:**
- GPS is turned OFF

**Solution:**
- Turn ON location services
- Allow location permission for Rapido app

## Issue 3: No riders available
**Possible reason:**
- High demand

**Solution:**
- Wait for 5 minutes and try again
- Change pickup location slightly

## Issue 4: Payment failed
**Possible reason:**
- Network issue
- Bank server down

**Solution:**
- Retry payment
- Use cash option
